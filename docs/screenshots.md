# Recommended Screenshot Checklist

Capture these views to build a strong visual portfolio or LinkedIn post.

- [ ] **The "Full Context" View**: Upper Cook Inlet selected, Beluga Habitat toggled ON, and a community panel open. Shows the data fusion capability.
- [ ] **The "Drill-Down" View**: Southeast Alaska with the "Statistical Areas" layer active. Shows the complexity of fishery management lines.
- [ ] **The "Socioeconomic" View**: Cordova or Kodiak selected, showing the Census data panel (Population/Income) and the official red-dashed TIGER boundary.
- [ ] **The "Environmental" View**: Copper River USGS gage popup open, showing real-time streamflow numbers alongside the management district.
- [ ] **The "Search" View**: Search results dropdown open with a query like "Sitka" or "Salmon," showing the grouped result types.
- [ ] **The "Comparison" View**: Comparison mode active between Bristol Bay and Southeast, showing the side-by-side scale cards.
- [ ] **The "Backend Proof" View**: A terminal window showing the `LOADED DATASETS` or `CACHE HIT` logs, proving the proxy architecture.
